# 9. 科学计数法

### 概念

科学计数法是一种**计数方法**, 将一个数表示为 `a` 与 `10` 的 `n` 次方相乘的形式 (`1 ≤ |a| ＜ 10`, `a` 不为分数形式, 且 `n` 为整数)

如:

1. `1000` -> `1 * 10 * 10 * 10` -> `1*10^3` -> `1e3`

> 如想深入了解可参考: [CSDN](https://blog.csdn.net/diaomo9737/article/details/101724507)

### 示例

```cpp
#include <iostream>
using namespace std;

int main() {

	float f1 = 3.3e-2;
	cout << f1 << endl;

	float f2 = 3.3e2;
	cout << f2 << endl;

	return 0;
}
```

```output
0.033
330
```
