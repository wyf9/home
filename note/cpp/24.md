# 24. 三目运算符

- 单目运算符: `!`
- 双目运算符: `+`, `-`, `*`, `/`, `%`, `!=`, `>`, `<`, `>=`, `<=`, `&&`, `||`
- 三目运算符: `表达式? 真值 : 假值`

### 使用

语法: `表达式? 真值 : 假值`

?>如果表达式成立则返回真值, 表达式不成立则返回假值 (和 `if` `else` 类似)

### 示例

```cpp
#include <iostream>
using namespace std;
int main() {
    int a = 10;
    int b = (a < 5) ? 1 : 0;
    cout << "a = " << a << endl;
    cout << "b = " << b << endl;
    return 0;
}
```

```output
a = 10
b = 0
```
