# 17. 算术运算符 - 加减乘除

| 符号 | 术语 | 示例 | 结果 |
| --- | --- | --- | --- |
| `+` | 正 | +3 | 3 |
| `-` | 负 | -3 | -3 |
| `+` | 加 | 10 + 5 | 15 |
| `-` | 减 | 10 - 5 | 5 |
| `*` | 乘 | 10 * 5 | 50 |
| `/` | 除 | 10 / 5 | 2 |

### 使用

```cpp
cout << 1 + 2;
```

```output
3
```

##### 注意

1. 整数除以整数时, 结果默认会舍弃小数部分, 如 `3 / 2` 会得到 `1`

> 可以在值前加上 `(float)` / `(double)`, 强制转换为浮点型, 即可得到带小数的结果: `3 / 2` -> `1.5`

2. **除数不能为 `0`**

### 示例

```cpp
#include <iostream>
using namespace std;
int main() {
    
    int a = 20, b = 10;
    cout << "a + b = " << a + b << endl;
    cout << "a - b = " << a - b << endl;
    cout << "a * b = " << a * b << endl; // SHIFT + 8
    cout << "a / b = " << a / b << endl;
    cout << "b / a = " << b / a << endl; // 整 / 整 = 整, 舍弃小数, 则为为 0
    cout << "(float)b / a = " << (float)b / (float)a << endl; // 强制类型转换 (int -> float / double)

    int c = 10, d = 0;
    cout << "c / d = " << c / d << endl; // 不能除以 0

    return 0;
}
```

a - b = 10
a * b = 200
a / b = 2
b / a = 0
