# 1-31. 求解表达式 (a+b)/c 的值

**程序名**: 求解表达式 (a+b)/c 的值

**题目描述**: 输入三个整数，分别为 `a`, `b`, `c` 的值，求 `(a+b)/c` 的值

**输入**: 三个整数 `a`, `b`, `c`

**输出**: 计算结果 **(保留 1 位小数位)**

**样例输入**:
```text
1 2 3
```

**样例输出**:
```text
1.0
```

### 代码

<details>
<summary>点击展开</summary>

```cpp
#include <iostream>
using namespace std;
int main()
{
    // 1. 定义变量
    int a, b, c;
    float d;
    // 2. 输入数据
    cin >> a >> b >> c;
    // 3. 计算或处理数据
    d = (a + b) / (float)c; // 整数除以整数时, 结果仍然为整数 -> 强制类型转换 -> 浮点数
    // 4. 输出结果
    printf("%.1f\n", d); // 保留 1 位小数
    return 0;
}
```

```output
1 2 3
1.0
```
