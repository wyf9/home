# 1-3. 变量的定义

变量，即会变的量，一般用来存储程序中的临时数据，可以修改，需要先定义后使用 (也可在定义的同时赋值)

#### 定义

1. 数据类型 变量名 *(=初始值)*;
2. 数据类型 变量1 *(=初始值)*, 变量2 *(=初始值)*, 变量3 *(=初始值)*......, 变量n *(=初始值)*;

可以在变量定义时就进行赋值，也可以在定义后再赋值:

```cpp
int a;
a = 123;
```

等同于:

```cpp
int a = 123;
```

#### 使用

使用时不加引号

```cpp
int a = 123;
cout << "a" << " = " << a;
```
```output
a = 123
```

其中, `"a"` 和 `" = "` 都为字符串，按原样输出;

`a` 为变量名, 则输出了变量 `a` 的值 `123`


### 示例代码

```cpp
#include <iostream>
using namespace std;

int main() {
	/*
	变量的定义:
		1.数据类型 变量名=初始值;
		2.数据类型 变量1, 变量2, 变量3......, 变量n;
	*/

	int a = 6, b, c;
	b = 7;
	c = 8;
	cout << "a = " << a << ", b = " << b << ", c = " << c << endl;
	return 0;
}
```
