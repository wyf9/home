# 1-29. 扩展-C++ 万能头文件

### 日常使用的部分头文件

```cpp
#include <iostream> // 平时最常使用

#include <stdlib.h> // rand() / srand()

#include <time.h> // or:
#include <ctime> // time()
```

### 万能头文件

```cpp
#include <bits/stdc++.h>
```

?> 包含了所有的 C++ 标准头文件, 可直接替换相应的所有 `include` 语句

!> 比赛可用, 但国内*部分*OJ系统**不支持**
