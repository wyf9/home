# 2-4. 三只小猪称体重

练习案例：有三只小猪 `a`, `b`, `c`，分别输入三只小猪的体重，并且判断哪只最重，输出最重的体重值

### 分析

比较 `a` 和 `b`，有三种可能情况：

| 情况      | 再次判断:    |
| --------- | ------------ |
| `a` < `b` | `b`, `c`     |
| `a` > `b` | `a`, `c`     |
| `a` = `b` | `a`/`b`, `c` |

```cpp
// `m` 存储最终结果
if (a < b) { // 1.
    if (c > b) {
        m = c;
    } else {
        m = b;
    }
} else if (a > b) { // 2.
    if (c > a) {
        m = c;
    } else {
        m = a;
    }
} else { // 3.
    // a==b
    if (c > a) {
        m = c;
    } else {
        m = a; // a == b, 都为最大值
    }
}
```

### 示例

```cpp
#include <iostream>
using namespace std;

int main() {
    float a, b, c, m;
    cout << "请输入三个数: ";
    cin >> a >> b >> c;
    if (a < b) { // 1.
        if (c > b) {
            m = c;
        } else {
            m = b;
        }
    } else if (a > b) { // 2.
        if (c > a) {
            m = c;
        } else {
            m = a;
        }
    } else { // 3.
        // a==b
        if (c > a) {
            m = c;
        } else {
            m = a; // a == b, 都为最大值
        }
    }
    cout << "最大的体重是: " << m << endl;
    return 0;
}
```
