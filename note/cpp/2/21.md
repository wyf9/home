# 2-21. 计算星期几

**程序名**: 计算星期几

**题目描述**: 输入两个整数 `a`, `b`, 计算 `a*b` 天后是星期几 **(假设今天是星期日)**，并按下列表格执行:

| **如果是** | **则输出** |
| ---------- | ---------- |
| 星期一     | `Mon`      |
| 星期二     | `Tue`      |
| 星期三     | `Wen`      |
| 星期四     | `Thr`      |
| 星期五     | `Fri`      |
| 星期六     | `Sat`      |
| 星期日     | `Sun`      |

**输入**: 两个整数 `a`, `b`

**输出**: `Mon` / `Tue` / `Wen` / `Thr` / `Fri` / `Sat` / `Sun`

**样例输入**:
```text
2 8
```

**样例输出**:
```text
Tue
```
