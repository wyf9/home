# 2-11. 判断是否是两位数, 判断两个数大小

### 1. 判断是否是两位数

**程序名**: 判断是不是两位数

**题目描述**: 输入一个数,判断这个数是不是一个两位数 *(10~99)*,如果是输出 1,不是输出 0

**输入**: 一个整数 `n`

**输出**: 1 或 0

**样例输入**:
```text
15
```

**样例输出**:
```text
1
```

```cpp

}
```

### 2. 判断两个数大小

**程序名**: 判断两个数的大小

**题目描述**: 输入两个 float 类型的数 `x`, `y`,比较两个数的大小,如果`x>y`输出1, 如果`x=y`输出0, 如果`x<y`输出2

**输入**: 两个 float 类型的数 `x`, `y`

**输出**: 1 / 0 / 2

**样例输入**:
```text
5 6
```

**样例输出**:
```text
2
```

```cpp
} else {
    cout << 2 << endl;
}
```

### 完整示例

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "-----1-----" << endl;
    int n;
    cin >> n;
    if (n >= 10 && n <= 99) {
        cout << 1 << endl;
    } else {
        cout << 0 << endl;
    }

    cout << "-----2-----" << endl;
    float x, y;
    cin >> x >> y;
    if (x == y) {
        cout << 0 << endl;
    } else if (x > y) {
        cout << 1 << endl;
    } else {
        cout << 2 << endl;
    }

    return 0;
}
```
