# 2-20. 判断成绩是否优秀

**程序名**: 判断成绩是否优秀

**题目描述**: 输入五门课的成绩，满足以下三个条件之一的都认为是成绩优秀:

- 总分 >= `450`
- 每门课高于 `88` 分
- 前三门课高于 `95` 分, 后两门高于 `80` 分

**输入**: `5` 个整数，表示五门课的成绩

**输出**: `优秀` / `不优秀`

**样例输入**:
```text
95 95 95 95 95
```

**样例输出**:
```text
优秀
```

### 程序示例

<details>
<summary>点击展开</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b, c, d, e;
    cin >> a >> b >> c >> d >> e;
    bool b1 = a + b + c + d + e >= 450;
    bool b2 = a > 88 && b > 88 && c > 88 && d > 88 && e > 88;
    bool b3 = a > 95 && b > 95 && c > 95 && d > 80 && e > 80;
    if (b1 || b2 || b3) {
        cout << "优秀" << endl;
    } else {
        cout << "不优秀" << endl;
    }
    return 0;
}
```

> 将 3 个条件单独拆分为布尔值进行计算
