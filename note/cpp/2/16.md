# 2-16. 求平方根

**程序名**: 平方根 (开方)

**题目描述**: 输入一个数，求这个数的平方根

**输入**: 一个数

**输出**: 这个数的平方根

**样例输入**:
```text
4
```

**样例输出**:
```text
2
```

### 平方根 (开方)

平方: `5^2` = `25`, `6^6` = `36`

开方: 平方的**逆运算**

即:

`25` 的开方 -> 哪个数的平方等于 `25` -> `5`
`25` 的开方 -> 哪个数的平方等于 `36` -> `6`

?> 求开方的函数: `sqrt()` (先导入 `math.h` / `cmath`)

!> `sqrt()` 的结果可能为小数!!!
