# 2-22. 温度换算

**程序名**: 温度换算 (摄氏华氏)

**题目描述**: 利用公式 `C=5*(F-32)/9` (`C` 为摄氏, `F` 为华氏), 进行计算转化

**输入**: 华氏温度 *(float/double 小数)*

**输出**: 摄氏温度 **(精确到小数点后 `5` 位)**

**样例输入**:
```text
100
```

**样例输出**:
```text
37.77778
```

### 程序示例

<details>
<summary>点击展开</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    double c, f;
    cin >> f;
    c = 5 * (f - 32) / 9;
    printf("%.5f", c);
    return 0;
}
```
