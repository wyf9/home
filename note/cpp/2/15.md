# 2-15. 四舍五入计算

**程序名**: 四舍五入

**题目描述**: 输入一个**浮点数**，对该数进行四舍五入计算

**输入**: 一个**浮点数**

**输出**: 四舍五入后的数

**样例输入**:
```text
1.5
```

**样例输出**:
```text
2
```

?> 四舍五入函数: `round()` (需数学函数库)

### 代码示例

<details>
<summary>点击展开</summary>

```cpp
#include <cmath>
#include <iostream>
// #include <bits/stdc++.h>
using namespace std;

int main() {
    float f;
    cin >> f;
    cout << round(f);

    return 0;
}
```
