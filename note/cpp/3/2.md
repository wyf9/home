# 3-2. 猜数字

案例: 系统随机生成一个 `1` 至 `100` 之间的数字,玩家进行猜测, 如果猜错, 提示玩家数字过大或过小, 如果猜对恭喜玩家胜利. 并退出游戏.

### 思路

1. bool `flag` = `true`;
2. 生成一个随机数;
3. while (`flag` == `false`) {
    - 用户输入数字;
    - if 比较正确: {
        - 恭喜玩家;
        - 跳出循环 - `flag` = `false`;
    }
    - else *[不正确]*: {
        - 输出 `大了` 或 `小了`;
    }
}
1. 退出
